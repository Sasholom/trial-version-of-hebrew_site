# 🚀 SaSholom AI

AI-портал со **100 экспертными персонажами**: чат с GPT/Gemini, анализ фото (Claude vision),
голосовой ввод и озвучка, загрузка PDF/TXT, три языка интерфейса (русский, English, עברית с RTL),
PWA с офлайн-режимом.

**Продакшен:** https://sasholomsite.vercel.app

---

## Быстрый старт

### Посмотреть интерфейс локально (без AI)

Нужен любой статический сервер, например:

```bash
npm run serve        # python3 -m http.server 8080
```

Открой http://localhost:8080. Всё, кроме ответов AI, работает.

### Полноценный запуск с AI

Нужен [Vercel CLI](https://vercel.com/docs/cli) и ключ Chad API:

```bash
vercel env add CHAD_API_KEY   # один раз: добавить ключ
npm run dev                   # vercel dev — статика + serverless-функция
```

### Деплой

Пуш в подключённый к Vercel репозиторий — деплой автоматический.
Единственная переменная окружения: `CHAD_API_KEY` (Vercel → Project → Settings → Environment Variables).

## Как это работает

1. Пользователь выбирает персонажа (или «Обычный» режим) и пишет вопрос.
2. `main.js → askAI()` собирает запрос: вопрос + системный промпт персонажа +
   заметки о пользователе + последние 10 сообщений контекста (+ фото, если прикреплено).
3. `api/chat.js` на сервере проксирует запрос в Chad API
   (текст → `gpt-5-nano`/`gemini-3-flash`, фото → `claude-3.7-sonnet`).
4. Ответ «печатается» по словам, затем рендерится как Markdown с подсветкой кода.
5. История и настройки хранятся в localStorage.

## Частые задачи

| Задача | Где |
|---|---|
| Добавить персонажа | [js/characters.js](js/characters.js) — добавь объект в массив; категория и селекты обновятся сами |
| Добавить перевод / язык | [js/i18n.js](js/i18n.js) — `translations` и `categoryNames`; в HTML тексты помечаются `data-i18n` |
| Поменять модели AI | [api/chat.js](api/chat.js) — `MODEL_ENDPOINTS` |
| Поменять лимиты (длина вопроса, контекст, размер фото) | [js/config.js](js/config.js) и `MAX_QUESTION_LENGTH` в [api/chat.js](api/chat.js) |
| Изменить статику | Не забудь поднять версию `CACHE_NAME` в [sw.js](sw.js), иначе у пользователей останется старый кэш |

## Известные ограничения

- **Rate limiting** (20 запросов/мин с IP) — in-memory, на serverless не разделяется между
  инстансами и сбрасывается при холодном старте. Для строгого лимита нужен Vercel KV/Upstash.
- **История** живёт в localStorage — очищается вместе с данными браузера, между устройствами
  не синхронизируется.
- Голосовой ввод — Web Speech API, стабильно работает только в Chrome/Edge.

## Дорожная карта (кратко)

Уникальные механики «коллективного разума» — в порядке реализации:
1. **Авто-подбор персонажа** — анализ вопроса → выбор лучшего эксперта.
2. **Совет директоров** — один вопрос → три параллельных ответа разных персонажей.
3. **Спор персонажей** — два персонажа дебатируют на заданную тему.
4. **Консилиум** — 5+ персонажей + AI-агрегатор с итогом.

Полный бэклог — в проектном брифе.
